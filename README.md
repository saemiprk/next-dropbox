### Next + TypeScript

Next의 활용 방법을 익히기 위하여 진행한 프로젝트 입니다.

이 프로젝트를 통해 Next와 Supabase의 Storage를 활용한 CRUD를 구현 하는 방법에 대해 공부할 수 있었고, 

기존에는 recoil이나 redux로 상태관리를 했었지만, Next를 시작하면서 접하게 된 react-query로 상태관리 하는 방법도 익힐 수 있었습니다.
